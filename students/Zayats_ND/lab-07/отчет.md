### 3. Event-Driven Sync

**События:**
- `MovieCreatedEvent` → создать сираницу Movie
- `WatchlistItemAddedEvent` → добавить в Watch list фильм

**Код:**
```java
public class MovieCreatedEvent {

    private final MovieReadJpaRepository repo;

    public MovieCreatedEvent(MovieReadJpaRepository repo) {
        this.repo = repo;
    }

    public void on(MovieCreatedEvent e) {

        MovieReadEntity m = new MovieReadEntity();

        m.setId(e.getMovieId());
        m.setTitle("Unknown");
        m.setGenre("Unknown");
        m.setRating(0.0);
        m.setDescription("Created via event");

        repo.save(m);
    }
}
```
